# Changelog

Все изменения проекта «Калькулятор смет» для ИТА Сочи.

## [1.1.0] - 26 февраля 2026

### Добавлено
- **Экспорт в PDF**
  - Генератор PDF на основе Dompdf
  - Профессиональный дизайн сметы
  - Автоматическое скачивание файла
  - Форматирование с сохранением стилей
  
- **Экспорт в Excel**
  - Генератор XLSX на основе PhpSpreadsheet
  - Два листа: "Смета" и "Исходные данные"
  - Форматирование ячеек и заголовков
  - Автоматическая ширина колонок

- **API Endpoints**
  - `POST /api/generate-pdf` - генерация PDF
  - `POST /api/generate-excel` - генерация Excel

- **Дизайн и UI**
  - SVG изображения для категорий (СКС, Видео, СКУД, IT, Электро)
  - Анимация при наведении на карточки
  - Увеличенные изображения категорий (320x240)
  - Эффект масштабирования изображений
  - Glow-эффект для итоговой суммы
  - Skeleton loading для загрузки
  - Ripple effect для кнопок
  - Toast уведомления с анимацией
  - 3D эффект для карточек
  - Плавающая анимация для иконок

- **CSS улучшения**
  - Улучшенные hover эффекты для карточек
  - Анимация gradient border
  - Кастомные стили для scrollbar
  - Стили для toast уведомлений
  - Анимации fade in, slide in, success

### Изменено
- Обновлен `calculator.js` с функциями экспорта
- Обновлен `calculator.css` с новыми анимациями
- Обновлен `index.html` с новыми классами для изображений
- Обновлен `INSTALL.md` с инструкцией по установке Composer

### Исправлено
- Мелкие баги в верстке таблицы результатов
- Оптимизирована загрузка JavaScript

### Технические детали
- Добавлен `composer.json` для управления зависимостями
- Созданы классы `PdfGenerator.php` и `ExcelGenerator.php`
- Обновлен `api/index.php` с новыми endpoints

## [1.0.0] - 26 февраля 2026

### Добавлено
- **Первая версия MVP**

- **База данных**
  - 8 таблиц (categories, templates, materials, works, template_items, coefficients, coefficient_values, template_coefficients)
  - 6 шаблонов (СКС, Видеонаблюдение, СКУД, IT-аутсорсинг, 2 электромонтажа)
  - 5 коэффициентов (срочность, территория, тип стен, высота, тип проводки)
  - Система объемных скидок (5% и 10%)
  - Логирование расчетов

- **Backend (PHP API)**
  - REST API endpoints:
    - `GET /api/categories`
    - `GET /api/templates`
    - `GET /api/templates/{id}`
    - `GET /api/category/{slug}/templates`
    - `POST /api/calculate`
    - `GET /api/additional-works`
  - Класс Database (PDO)
  - Модель CalculatorModel

- **Frontend**
  - Главная страница с категориями
  - Страница выбора шаблонов
  - Интерфейс калькулятора
  - Ползунок для ввода параметров
  - Выбор коэффициентов (радио-кнопки)
  - Дополнительные работы
  - Таблица результатов
  - Кнопки печати

- **Дизайн**
  - Адаптивный дизайн (десктоп, планшет, мобильный)
  - Tailwind CSS
  - Font Awesome иконки
  - Кастомные стили для form elements

- **Документация**
  - README.md
  - INSTALL.md
  - SQL скрипт с комментариями
  - config.example.php

- **Интеграция**
  - Ссылки в главном меню сайта
  - Мобильное меню

## Формат версий

Проект использует [Semantic Versioning](https://semver.org/lang/ru/):
- **MAJOR** (1.0.0) - значительные изменения, несовместимые с предыдущей версией
- **MINOR** (1.1.0) - новая функциональность, совместимая с предыдущими версиями
- **PATCH** (1.1.1) - исправления багов, совместимые с предыдущими версиями

## Формат дат

Используется формат ISO 8601: `ГГГГ-ММ-ДД`
