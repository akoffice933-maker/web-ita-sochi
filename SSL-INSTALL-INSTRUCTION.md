## 🔐 ISPManager: Установка SSL

### Шаг 1: Зайдите в панель

```
Адрес: https://school-board.online:1500
Логин: infoitaso3_board
Пароль: HGKHGJFGasadsdf12##
```

### Шаг 2: Создайте сертификат

1. **Меню слева:** WWW → SSL-сертификаты
2. **Кнопка:** "Создать"
3. **Тип:** Let's Encrypt
4. **Домен:** `school-board.online`
5. **Email:** ваш email
6. **Нажмите:** "OK"
7. **Ждите:** 5-10 минут

### Шаг 3: Привяжите к домену

1. **Меню:** WWW → WWW-домены
2. **Найдите:** `school-board.online`
3. **Кнопка:** "Изменить" (карандаш)
4. **Вкладка:** "SSL"
5. **Выберите:** созданный сертификат
6. **Нажмите:** "OK"

### Шаг 4: Проверьте

```
https://school-board.online/
```

## 🔐 cPanel: Установка SSL

### Шаг 1: Зайдите в панель

```
Адрес: https://school-board.online:2083
Логин: infoitaso3_board
Пароль: HGKHGJFGasadsdf12##
```

### Шаг 2: Найдите SSL

1. **Раздел:** "Безопасность" (Security)
2. **Нажмите:** "SSL/TLS" или "Let's Encrypt SSL"

### Шаг 3: Установите

1. **Кнопка:** "Manage SSL sites"
2. **Домен:** `school-board.online`
3. **Кнопка:** "AutoSSL" или "Install Let's Encrypt"
